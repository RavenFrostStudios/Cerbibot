# Website Brief

Updated: 2026-03-21

## Goal

Give the website builder a clear, accurate description of what CerbiBot is now, without overpromising channel breadth or ecosystem maturity.

## One-Sentence Product Description

CerbiBot is a local, security-first orchestration platform for running multi-mode AI workflows with governed skills, project-scoped memory, provider controls, and a live dashboard.

## Short Homepage Version

Run a self-hosted AI control plane with multi-mode orchestration, governed tool use, project isolation, artifacts, audit logging, and a dashboard built for real operator workflows.

## Core Messaging Pillars

### 1. Orchestration depth
- More than a single chat box
- Multiple collaboration modes for different task shapes
- Runs, checkpoints, heartbeats, blockers, and recovery-aware execution

### 2. Governance and safety
- Approval-aware tool execution
- skill manifests, validation, signing, and verification
- audit and export controls
- admin-password-gated sensitive actions

### 3. Local operator control
- self-hosted daemon
- dashboard + CLI
- provider config and diagnostics
- project-scoped sessions and memory

### 4. Practical workflow support
- skills
- artifacts
- webhook triggers
- delegation patch workflow
- MCP configuration and health checks

## What To Avoid Saying

Do not imply:
- full OpenClaw-style messaging channel coverage
- broad consumer-ready companion app ecosystem
- polished enterprise SaaS positioning
- that every planned connector/integration already exists

Do say:
- technical preview if needed
- self-hosted/local-first
- operator-facing
- orchestration and governed execution
